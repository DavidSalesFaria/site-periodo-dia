# Site Período do dia (manhã, tarde, noite)

O site muda a sua aparência de acordo com os períodos do dia.

**Manhã:** Períodos entre 00:00 e 11:59 

**Tarde:** Períodos entre 12:00 e 17:59

**Noite:** Períodos entre 18:00 e 23:29
